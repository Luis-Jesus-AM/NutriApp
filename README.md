# ¿Qué funcionalidad puede tener una app nutricional?

El objetivo de una app nutricional es principalmente ayudar a mejorar tus hábitos alimenticios y proporcionarte una mejor calidad de vida, además de ayudar a alcanzar objetivos con la salud (puede ser para ayudarte a prevenir o controlar enfermedades cardiovasculares), el peso y/o rendimiento físico, las funciones que pueden tener estas apps son las siguientes:
## Registrar lo que comes
•	Puedes anotar o escanear lo que consumes y así calcular las calorías y macronutrientes (proteínas, carbohidratos, grasas) y, a veces, micronutrientes (vitaminas, minerales).

## Alcanzar tus objetivos personal
•	Perder peso, ganar peso, ganar masa muscular o solo mantenerte en forma y mejorar tu salud.
•	Algunas apps pueden incluso crearte un plan alimenticio. 
## Planificación de menús
•	Generar planes de alimentación personalizados adaptándose las necesidades de cada usuario.
•	Sugerencias de recetas saludables.
## Evaluación nutricional
•	Calcular el IMC, TMB, gasto energético.
•	Analizar si la alimentación cumple con los requerimientos nutricionales.
•	Test para evaluar hábitos alimenticios.
## Educación alimenticia
•	Consejos diarios de alimentación. 
•	Mitos y verdades sobre los alimentos.
## Función extra para nutricionistas
•	Panel de pacientes.
•	Hacer seguimiento del progreso del paciente.
•	Chat o video llamada para consultas online.

# Los datos que necesita una app nutricional inicialmente son muy básicos, como, datos personales:
•	*Nombre o apodo.*
•	*Edad.*
•	*Sexo.*
•	*Altura.*
•	*Peso actual.*
•	*Peso objetivo.*
•	*Nivel de actividad física* (sedentario, moderado, activo).
•	*Fecha de nacimiento* (opcional, para cálculos precisos).
•	*Objetivo principal* (perder peso, mantener, ganar masa muscular).

## También puede requerir otros datos como, datos de salud:
•	*Condiciones médicas* (diabetes, hipertensión, colesterol, etc.).
•	*Alergias alimentarias* (frutos secos, lácteos, gluten, etc.).
•	*Intolerancias* (lactosa, gluten).
•	*Medicamentos que esté tomando* (si afectan al metabolismo o apetito).

## Preferencias alimenticias:
•	*Dieta seguida* (vegetariana, vegana, keto, paleo, etc.).
•	*Alimentos que no le gustan o que quiere evitar.*
•	*Horarios habituales de comida.*
